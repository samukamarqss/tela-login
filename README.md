## 📌 Sobre o Projeto

Este projeto consiste em uma tela de autenticação web com um design limpo e agradável. A interface foi construída pensando na usabilidade, oferecendo campos claros para inserção de credenciais e um feedback visual após o clique no botão de login, levando à tela de "Login realizado com sucesso!".

A identidade visual conta com uma imagem de farol e barco que transmite tranquilidade, contrastando perfeitamente com a paleta de cores neutras e os botões em tons de roxo/azul. O grande destaque é a construção de todo o layout e fluxo visual **sem a utilização de JavaScript**.

## ✨ Funcionalidades

- [x] Interface de login intuitiva e moderna.
- [x] Campos de input estilizados.
- [x] Transição de estado (Tela de Login ➡️ Tela de Sucesso) feita puramente com HTML/CSS.
- [x] Layout responsivo e adaptável.
- [x] Efeitos de hover e transições suaves nos botões.

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido com foco em estilização e marcação semântica:

* **HTML5:** Estruturação semântica do conteúdo e links entre as telas.
* **CSS3:** Estilização avançada, layout (Flexbox/Grid), variáveis de cores e efeitos visuais.
