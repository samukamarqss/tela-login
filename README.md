## 📌 Sobre o Projeto

Este projeto consiste em uma tela de autenticação web com um design limpo e agradável. A interface foi construída pensando na usabilidade do usuário, oferecendo campos claros para inserção de credenciais e um feedback visual imediato após o clique no botão de login, alternando para uma mensagem de "Login realizado com sucesso!".

A identidade visual conta com uma imagem de farol e barco que transmite tranquilidade, contrastando perfeitamente com a paleta de cores neutras e os botões em tons de roxo/azul.

## ✨ Funcionalidades

- [x] Interface de login intuitiva e moderna.
- [x] Campos de input estilizados (Usuário e Senha).
- [x] Transição de estado (Tela de Login ➡️ Tela de Sucesso).
- [x] Layout responsivo e adaptável.
- [x] Efeito de hover nos botões.

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as tecnologias fundamentais de desenvolvimento web:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout em Flexbox/Grid e transições visuais.
* **JavaScript:** Lógica de manipulação do DOM para simular a autenticação e alterar as telas.
